# Day 5 ― Git / CSS / JavaScript基礎

## Git コマンド
- `git add .` → 変更ファイルをまとめる
- `git commit -m "メモ"` → 変更内容を記録する
- `git push` → GitHubにアップロード
- 1行でまとめる：`git add . && git commit -m "変更内容" && git push`

## CSS
- `display: grid` + `grid-template-columns` → カードを横並びにする
- `transition` → 変化をなめらかにする
- `:hover` → マウスを乗せたときのスタイル
- `:active` → クリックした瞬間のスタイル
- `linear-gradient` → グラデーション背景
- `opacity` + `transform: translateY` → 透明にして下にずらす

## JavaScript
- `if (要素)` → その要素が存在するときだけ動かす
- `classList.add / replace` → クラスを追加・入れ替える
- `textContent` → JSから文字を変える
- `IntersectionObserver` → 画面に入ってきたことを検知する
- `setTimeout(関数, ミリ秒)` → 時間差で処理を実行する

## 外部ライブラリ
- Google Fonts → フォントを読み込む
- Font Awesome → アイコンを使う（`<i class="fa-...">` で表示）
